# EAN CLI
Tool to discover leaked API tokens in JavaScript files

![Screenshot of EAN CLI](https://i.imgur.com/Zhw6YhG.png)


# Install
`pip3 install -r requirements.txt`



# Usage
> python3 tokens.py [domain] [flags]


# Syntax
-u [URL], --url [URL]
> URL to scan

-m [n], --max [n] (optional)
> Max number of tokens to print
